Python 3.13 or later

numpy library

pandas library

yfinance library

matplotlib library

Open command line > pip install numpy pandas yfinance matplotlib

Save the files provided in the following structure

Root folder: Stock Tracker by Mattias.py, stock_search.py, stock_charts.py

Run Stock Tracker by Mattias.py
